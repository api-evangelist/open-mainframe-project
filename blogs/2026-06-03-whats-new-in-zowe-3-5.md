Zowe 3.5 introduces updates to streamline enterprise development and operations, with the CLI approaching half a million downloads and Explorer for VS Code reaching approximately 238,000 active users. Key enhancements include API Mediation Layer improvements with separate server and client certificate support, CLI upgrades for secure certificate imports and faster SSH execution, and Explorer for VS Code additions like multi-member search patterns and job polling. The CICS extension gains tree state preservation during refreshes and resource comparison across regions, while the next version targets General Availability for Zowe Remote SSH and Node.js modernization.
