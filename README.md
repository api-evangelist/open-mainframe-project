# Open Mainframe Project (open-mainframe-project)
The Open Mainframe Project is a Linux Foundation project encouraging the use of Linux-based operating systems and open source software on mainframe computers. Founded in 2015 with IBM, it hosts projects like Zowe for modern interfaces to z/OS and promotes mainframe skills development.
