# Open Mainframe Project (open-mainframe-project)

The Open Mainframe Project is a Linux Foundation project encouraging the use of Linux-based operating systems and open source software on mainframe computers. Founded in 2015 with IBM, it hosts projects such as Zowe (modern interfaces for z/OS), Feilong (z/VM cloud connector), Galasa (testing), and a range of community programs that promote mainframe skills and open source on IBM Z and LinuxONE platforms.
